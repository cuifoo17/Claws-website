About Usage

Only two things draw down your usage. 1) Messaging Claws and 2) Using the voice readout feature. Longer messages use more than shorter ones. Additionally, if Claws accesses the internet to answer a question, this specific message will cost 3-4 times more than a normal message. As for the voice readout, each readout costs as much as an internet search message, so you might see the read out draw down usage rather quickly if used on every message. I've optimized this system as much as possible so you can get the most bang for your buck, if you have more questions, please email support with the title "Flamingos are pink" and i'll get back to you.

Every tier gives you the exact same Claws — the same companion, the same features. The only thing that changes as you move up is how much usage you get: higher tiers simply give you more room to chat.

Claws's memory also runs quietly in the background so he remembers what matters to you between conversations. Some of that happens when you first open the app, which is why your usage can tick up a little before you've sent anything. I've tried to lower this cost as much as possible in the backend, it's as cheap as current technology allows. If you're actually reading this, thanks for using Claws :).
